「AがBに最も近い整数である」は、「Bを四捨五入するとAになる」
とか「|A-B|<0.5」ということらしい。

~~~
問題文より、
φ = (1 + √5) / 2
ψ = (1 - √5) / 2
黄金比の定義より、
φ^2 = φ + 1
また、
ψ^2 = ψ + 1

n = 0のとき
Fibonacci数の定義より、
Fib(0) = 0
（φ^0 - ψ^0) / √5 = 0
よって、n = 0のとき、Fib(n) = (φ^n - ψ^n) / √5が成り立つ。

n = 1のとき
Fibonacci数の定義より、
Fib(0) = 1
（φ^1 - ψ^1) / √5 = ((1 + √5) - (1 - √5)) / 2 = 1
よって、n = 0のとき、Fib(n) = (φ^n - ψ^n) / √5が成り立つ。


n = a - 1, n = a - 2のとき、Fib(n) = (φ^n - ψ^n)/ √ 5 が成り立つと仮定する。（aは3より大きい整数）
n = aのとき、
Fibonacci数の定義より、
Fib(a) = Fib(a - 1) + Fib(a - 2)
= (φ^(a-1) - ψ^(a-1)) / √5 + (φ^(a-2) - ψ^(a-2)) / √5
= ((φ^(a-1)　+ φ^(a-2)) - (ψ^(a-1) + ψ^(a-2))) / √5
= (φ^(a-2) * (φ + 1) - (ψ^(a-2) * (ψ + 1)) / √5
黄金比の定義より
= (φ^(a-2) * (φ^2) - (ψ^(a-2) * (ψ^2))) / √5
= (φ^a - ψ^a) / √5
よって、n = aのとき、Fib(n) = (φ^n - ψ^n) / √5が成り立つ。

以上より、 n >= 0のとき、(φ^n - ψ^n) / √5が成り立つ。

Fib(n) = (φ^n - ψ^n) / √5
Fib(n) - φ^n / √5 = -ψ^n / √5
ここでψ = (1 - √5) / 2 より
-1 < ψ = (1 - √5)/2 < -1/2
|-1| > |ψ^n| > |-1/2|
2 < √5なので、
1/2 > |-1|/√5 > |ψ^n|/√5 > |-1/2|/√5
よって、
|Fib(n) - φ^n / √5| = |-ψ^n / √5| < 1/2
より、φ = (1 + √5) / 2のとき、Fib(n)はφ^n/√5に最も近い整数。
~~~
